About

Lately I've been exploring interface design: when agents can do everything, do you need an interface at all?

I'm Will and I build products at the edge of new technology. Over the last ten years I've built personal assistants at Google, a consumer wallet at Uniswap, and a blockchain developer platform at Alchemy.

I'm building Architect, an interactive interface for Claude Plan Mode. The plugin generates a visual diagram so you can see the whole system at a glance and comment directly on nodes and edges to steer the plan.
